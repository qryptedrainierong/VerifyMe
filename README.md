
# VerifyMe Admin Portals

**VerifyMe** is a privacy-preserving **identity verification** platform: organizations integrate via OIDC-style flows, **VerifyMe Users** complete verification in the **mobile app**, and **Linked End Users** tie org records to those identities. This repository contains the **VerifyMe Admin Portal** and **Organization Admin Portal** front-end shells (Vite + React).
