# Documentation location

All VerifyMe product, UX, API, schema, maintenance, and engineering notes for this repository live under **[`../docs/`](../docs/README.md)** (`/docs` from the repo root).

Audit-specific specifications remain alongside other docs as `docs/audit-logs-*.md` (see the index in [`docs/README.md`](../docs/README.md)).

This `documentation/` folder exists only as a pointer for older links or bookmarks; do not add new long-form docs here.
